# 🌍 Carbon Storyteller: 同样的繁荣，不同的代价

本项目是针对全球主要经济体 2014-2023 年人均温室气体排放量的交互式探索。

## 📅 项目时间轴

*   **Day 1-2: 探索性数据分析 (EDA)**  
    对 OECD 原始 CSV 数据进行深入探索，识别出瑞典与澳大利亚等国家在排放强度上的极端差异。清洗数据中的聚合项，并处理缺失值。
*   **Day 3: 设计决策与原型**  
    对比了堆叠面积图、河流图与联动仪表盘方案。确定了以“比较+演变”为核心的叙事逻辑。
*   **Day 4-5: 核心逻辑开发**  
    使用 ECharts 构建联动系统。实现了点击柱状图实时同步折线图的底层数据流。开发了 DataZoom 缩放引擎以应对大量实体。
*   **Day 6: 叙事引擎与美化**  
    加入 Insights 动态洞察引擎，通过 Tailwind CSS 实现沉浸式深色模式 UI，并增加 OECD 平均值作为基准线。
*   **Day 7: 测试、部署与说明文档**  
    完成 GitHub Pages 部署，撰写网页集成版说明文档。

## 🔬 设计哲学

遵循交互设计金律：
1. **Overview First**: 第一眼展示全球排放概览。
2. **Zoom and Filter**: 通过搜索框和 DataZoom 快速定位目标国家。
3. **Details on Demand**: 点击柱条触发联动，查看特定国家的十年故事。

## 🛠️ 技术栈
ECharts 5.4.3, PapaParse, Tailwind CSS, JavaScript ES6+
